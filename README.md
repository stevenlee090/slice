# Slice

A bill-splitting app for groups. Track shared expenses, split costs equally or with custom amounts, and settle up with minimal transactions.

Live site: [stevenlee090.github.io/slice](https://stevenlee090.github.io/slice)

## Features

- Create sessions for trips or group events
- Add expenses with equal or custom splits
- Tag expenses by category (food, transport, accommodation, activities, entertainment)
- Automatic settlement calculation — minimises the number of payments needed
- Desktop-responsive layout with sidebar navigation and live right panel
- All data stored locally in the browser — no account required
- Archive sessions to keep a history

## Tech stack

- React + TypeScript
- Vite
- Tailwind CSS
- LocalStorage for persistence

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

The site is deployed to GitHub Pages via GitHub Actions on every push to `main`.
